# 十二支詣〜干支にまつわる神社探し〜

本リポジトリは、就職作品プレゼンテーション（2026）にて展示した  
個人制作作品 **「十二支詣」** の DEMO バージョンです。

## 開発構成

- `client/`: React + Vite フロントエンド（現行）
- `server/`: SQLite を読む API サーバー（`/api/*`）
- `data/`: JSON ソースと SQLite DB

## 起動方法

- 依存関係準備  
  `npm run setup:client`
- フル起動（推奨）  
  `npm run dev`  
  API (`http://localhost:8787`) と Vite (`http://localhost:5173`) を同時起動
- フロントエンドのみ  
  `npm run dev:client`
- API のみ  
  `npm run dev:server`
- 本番ビルド  
  `npm run build`


## 概要

> **十二支を辿りながら、自分と縁のある神社に出会おう**

- 神社巡りに興味はあるものの、  
始め方がわからない方や、自分と呼び合う神社を見つけたい方のためのサイトです

- 干支の動物とご利益を通して、  
  - 「神社探し」
  -  「十二支診断」
  -  「バーチャル参拝」
  -  「ことわざくじ」  
などの機能により、オンラインで神社巡りを楽しめる構成としています
- HAL東京進級制作展2024年度　銀賞受賞


## 制作背景

### 【課題】

- 神社の数は多いが、巡りに興味があっても始め方が分からない
- 情報サイトは多いものの、体験性が弱い

### 【解決策】

- 干支ｘご利益の明快な二重フィルターにより、「導かれる体験」を設計
- 診断や選択を重ねる構成により、能動的に参加できる体験をつくり出す


## セールスポイント

#### 01. モバイルフレンドリーな UI 設計

- 三カラムレイアウトを採用し、中央コンテンツへの没入感と各機能への即時アクセスを両立。
- レスポンシブ設計を前提としたレイアウト構成により、開発効率と体験の一貫性を両立。

#### 02. MPA＋SPAのハイブリッド構成

- React／Node.js によるハイブリッド構成を採用。
- SEOを意識したページ構造と、SPAによる滑らかな画面遷移を両立。

#### 03. 継続的に改善可能な構造

- 制作およびリニューアルを通じて、コンポーネント設計やイベント管理を整理。
- 更新や機能追加に対応しやすい、拡張性の高い構造へと段階的に改善。
